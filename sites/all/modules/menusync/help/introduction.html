Menu Sync open menu options based on the path being presented.

Menu Sync is similar to Menu Trails but it uses paths to define the pages affected.

Another important difference between Menu Sync and Menu Trails is the method used to set the desired menu option. The method used by Menu Sync allows it to affect more than one menu in the same page. Apparently this is not possible with Menu Trails.

Configuration

All configuration is done at admin/content/menusync.

The first step is to choose the menu that will be affected by the rule to be created. Click in the add rule link for the proper menu.

At the add rule page the first step is to choose the menu option that shall be activated by the rule.

The final mandatory step is to define the path of the page that should have the selected menu option activated. Wildcards are accepted at path definition.

The Description field may hold some kind of explanation for this rule.

As rules are checked in order and the first rules that is applied stops the process it might be important to change rule processing order with the Weight field.
